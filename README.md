# Mini Backend Project

Overview
This is a backend project built using Node.js, Express, and MySQL.
It includes user registration, login with JWT authentication, a protected dashboard, and CRUD operations.

  Tech Stack
- Node.js
- Express
- MySQL
- JWT Authentication
- bcrypt

  Features
- User Registration
- User Login
- Protected Dashboard Route
- CRUD operations (Create, Read, Update, Delete)

   How to Run
1. Install dependencies:
   npm install

2. Create a .env file with:
   DB_HOST=localhost
   DB_USER=your_mysql_user
   DB_PASSWORD=your_mysql_password
   DB_NAME=your_database_name
   JWT_SECRET=your_secret_key

3. Start the server:
   npm run dev

Server runs on:
http://localhost:5000
